Algoritmo A estrella

Repositorio github

Descripción

Se trata de una aplicación que simula un algoritmo A estrella.

Se permite realizar un recorrido por distintas posiciones predeterminadas denominadas "way points".

Se puede determinar celdas que son relativamente peligrosas, pero que pueden atravesarse asumiendo un cierto riesgo.

Da la posibilidad de varios vehículos en colaboración, en los que si una celda ha sido ya ocupada por uno quede prohibida para otro evitando colisiones.

Comentarios

La vista del programa consiste en 4 zonas: Cabecera, Plano, Iconos y Settings.

Cabecera: Para la transición con otras páginas como la de documentación.

Plano: Donde se realiza la simulación.

Iconos: Se encuentra todos los objetos que se pueden colocar.

None: Se usa para borrar objetos.

Bloque blanco: Son los obstáculos del plano.

Avatar: Inicio de la ruta

Bandera: Los sitios que hay que pasar. El orden es según lo creas.

Exclamación: Celdas peligrosas.

Settings: Para configuraciones.

La anchura y la altura determinan las dimensiones.

El botón de reset es para resetear todo a la configuración que está en el panel de settings.

Hay dos tipos de vistas: una vista aérea y otra perspectiva, se intercambian con el botón de vista.

El botón de ejecutar es para inicializar la simulación.

Zoom es el ratio de zoom in y zoom out.

Desplazamiento es el ángulo por el que gira la vista de la cámara.

Velocidad es la velocidad que se simula el camino.

Peligro es el porcentaje en el cual una celda peligrosa se pueda activar.

El botón para la subida de una imagen para cargar un plano con alturas (sin hacer).

Al hacer scroll en el ratón se puede realizar el zoom. También con la tecla espacio para zoom in, ctrl + espacio para zoom out.

Con el botón izquierdo del ratón se van asignando los objetos en el plano.

Con el botón derecho del ratón se cambia la posición de vista de la cámara. También se con las teclas de dirección del teclado.
